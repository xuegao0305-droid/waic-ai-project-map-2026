# WAIC 2026 AI 项目全量拆解

一个面向 WAIC 2026 官方项目目录的可交互行业研究网站

网站覆盖全部收录项目，并从一级行业、二级方向和具体产品三个层级拆解项目分布，同时提供行业热度、机器人任务、典型公司和项目查询等视图

## 在线访问

- GitHub Pages：<https://xuegao0305-droid.github.io/waic-ai-project-map-2026/>

```bash

```

